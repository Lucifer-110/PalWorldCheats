# 菜单特点

- 修改速度
- 修改攻击
- 修改防御
- 无限健康
- 无限耐力
- 无限弹药
- 显示玩家位置
- 将位置打印到控制台

# 功绩

- 安全传送
- 开放实体列表
- 给予经验值
- 传送回家
- 切换飞行
- 复活
- 物品生成器
- Easy Pal Condensation - 所有升级级别仅需要 1 Pal

# 传送器

- 所有Boss地点
- 自定义传送位置
- 传送到路径点

# 实体清单

- 显示附近实体的列表（玩家、NPC 和怪物）
- 玩家和/或Pal的过滤器列表
- 传送到选定的实体
- 杀死实体或设置为 1 HP
- 强制加入公会
- “Mask It”可用于您的玩家实体以隐藏玩家名称。

# 杂项

- 杀死光环
- 服务器崩溃 【暂时不可用】
- 解锁所有雕像
- 将 Pals 传送到 XHair/vacuum
- 解锁所有宝箱
